## 0. 사전 원칙 (모든 작업에 적용)

- 모든 UI 카피/인터랙션은 위 "헌법" 문장을 통과해야 함. 설득·비난·강제 금지.
- 파일 확장자 규칙: 화면 컴포넌트만 `.tsx`, 나머지(타입·유틸·훅·스토어·API 로직)는 `.ts`.
- 스타일: **inline style 유지** (Emotion/Tailwind 도입 금지). `cn` 헬퍼 없음.
- 한 파일 300줄 넘으면 반드시 분리. 주석은 "왜(Why)"만 한국어로.
- **MVP = 한 권 모드 단일**. 요일 루틴/인터리빙/모드 토글 UI는 모두 제거.
- 시간 표기는 **Pre-reading 단계에서만** ("보통 이 분량은 18분 정도"). Reading 중·완료 후는 시간 숨김.
- Streak/연속일수/목표 숫자 등 **정량 보상 UI는 어디에도 두지 않음**. 히트맵은 색 진하기만.
- **Next.js App Router**: AGENTS.md에 따르면 이 버전은 학습 데이터와 다를 수 있음 → 새 API(예: `params` async 등) 필요 시 `node_modules/next/dist/docs/`를 먼저 확인하고 코드 작성할 것.

### T-10: EMA pace 업데이트
**파일**: `src/utils/pace.ts` (신규)
**의존**: T-03
**변경 내용**:
- `updatePaceEMA(book, log): number | undefined`
  - `samples < EMA_MIN_SAMPLES` → `undefined` (숨김 상태)
  - 이상이면 `alpha * latest + (1-alpha) * prev`
- `T-05.commitLog` 안에서 호출해 `Book.avgMinPerPage` 갱신 (books store 별도 필요 시 T-05에서 같이 관리)

```ts
// src/utils/pace.ts
import { EMA_ALPHA, EMA_MIN_SAMPLES } from "@/constants/reading";

export function updatePaceEMA(opts: {
  prevPace: number | undefined;
  latestMinPerPage: number;
  sampleCount: number;          // 이 로그 포함 누적 횟수
}): number | undefined {
  const { prevPace, latestMinPerPage, sampleCount } = opts;
  if (sampleCount < EMA_MIN_SAMPLES) return undefined; // 3회 미만 예측 숨김
  if (prevPace == null) return latestMinPerPage;
  return EMA_ALPHA * latestMinPerPage + (1 - EMA_ALPHA) * prevPace;
}
```

### T-11: `useWakeLock` 훅
**파일**: `src/hooks/useWakeLock.ts` (신규)
**의존**: 없음
**변경 내용**:
- `navigator.wakeLock.request("screen")` 시도
- 실패(iOS 16 이하 등) 시 조용히 무시
- `visibilitychange`에서 `hidden`이면 release, 재활성 시 재요청
- return: `{ active: boolean, failed: boolean }`

```ts
// src/hooks/useWakeLock.ts
"use client";
import { useEffect, useRef, useState } from "react";

export function useWakeLock(enabled: boolean) {
  const [active, setActive] = useState(false);
  const [failed, setFailed] = useState(false);
  const sentinelRef = useRef<WakeLockSentinel | null>(null);

  useEffect(() => {
    if (!enabled) return;
    // iOS 16 이하 등 미지원 브라우저는 조용히 실패. 2분 자동 저장(T-22)이 안전망.
    const anyNav = navigator as any;
    if (!anyNav.wakeLock) { setFailed(true); return; }

    let cancelled = false;
    const acquire = async () => {
      try {
        const s = await anyNav.wakeLock.request("screen");
        if (cancelled) { s.release(); return; }
        sentinelRef.current = s;
        setActive(true);
        s.addEventListener("release", () => setActive(false));
      } catch {
        setFailed(true);
      }
    };
    acquire();

    const onVis = () => {
      if (document.visibilityState === "visible" && !sentinelRef.current) acquire();
    };
    document.addEventListener("visibilitychange", onVis);
    return () => {
      cancelled = true;
      document.removeEventListener("visibilitychange", onVis);
      sentinelRef.current?.release().catch(() => {});
      sentinelRef.current = null;
    };
  }, [enabled]);

  return { active, failed };
}
```

### T-12: `useVisibilityChange` (딴짓 감지)
**파일**: `src/hooks/useVisibilityChange.ts` (신규)
**의존**: 없음
**변경 내용**:
- Reading 화면에서 사용. 사용자가 앱을 떠났다가 돌아온 시점을 감지.
- 옵션: `minAwayMs` (예: 3000) — 미만이면 "우발적 터치"로 간주해서 무시
- callback: `onReturn(awayMs)`

```ts
"use client";
import { useEffect, useRef } from "react";

export function useVisibilityChange(opts: {
  enabled: boolean;
  minAwayMs?: number;
  onReturn: (awayMs: number) => void;
}) {
  const hiddenAtRef = useRef<number | null>(null);
  const minAwayMs = opts.minAwayMs ?? 3000;
  useEffect(() => {
    if (!opts.enabled) return;
    const onVis = () => {
      if (document.visibilityState === "hidden") {
        hiddenAtRef.current = Date.now();
      } else if (document.visibilityState === "visible" && hiddenAtRef.current) {
        const away = Date.now() - hiddenAtRef.current;
        hiddenAtRef.current = null;
        if (away >= minAwayMs) opts.onReturn(away);
      }
    };
    document.addEventListener("visibilitychange", onVis);
    return () => document.removeEventListener("visibilitychange", onVis);
  }, [opts.enabled, minAwayMs, opts.onReturn]);
}
```

### T-13: `useLongPress` 훅
**파일**: `src/hooks/useLongPress.ts` (신규)
**의존**: T-03
**변경 내용**:
- 0.8초 누르면 발화, 이동 10px 이상이면 취소
- progress 값(0~1) 반환 → 버튼 내부에 fill 링으로 시각화
- 터치/마우스 양쪽 지원

```ts
"use client";
import { useCallback, useRef, useState } from "react";
import { LONG_PRESS_MS } from "@/constants/reading";

export function useLongPress(onComplete: () => void) {
  const [progress, setProgress] = useState(0);
  const startRef = useRef<number | null>(null);
  const rafRef = useRef<number | null>(null);

  const start = useCallback(() => {
    startRef.current = Date.now();
    const tick = () => {
      if (startRef.current == null) return;
      const p = Math.min((Date.now() - startRef.current) / LONG_PRESS_MS, 1);
      setProgress(p);
      if (p >= 1) { cancel(); onComplete(); return; }
      rafRef.current = requestAnimationFrame(tick);
    };
    rafRef.current = requestAnimationFrame(tick);
  }, [onComplete]);

  const cancel = useCallback(() => {
    startRef.current = null;
    setProgress(0);
    if (rafRef.current) cancelAnimationFrame(rafRef.current);
  }, []);

  return {
    progress,
    handlers: {
      onMouseDown: start, onMouseUp: cancel, onMouseLeave: cancel,
      onTouchStart: start, onTouchEnd: cancel, onTouchCancel: cancel,
    },
  };
}
```

### T-37: Gemini Vision API route
**파일**: `src/app/api/extract-toc/route.ts` (신규)
**의존**: 없음
**변경 내용**:
- `POST multipart/form-data` — 이미지 1~N장
- 환경변수: `GEMINI_API_KEY`
- 모델: `gemini-2.5-flash` (비용·속도 균형)
- 요청: 이미지를 base64로 인라인 첨부
- 응답 스키마 (strict JSON):
  ```json
  {
    "parts": [
      {
        "index": 1,
        "title": "파트 1 · 원시의 기억",
        "startPage": 1,
        "endPage": 52,
        "sections": [
          { "title": "1장 · 사피엔스의 밤", "startPage": 1, "endPage": 28 }
        ]
      }
    ],
    "totalPages": 312,
    "confidence": 0.87
  }
  ```
- 프롬프트 템플릿 (고정):
  ```
  너는 책 목차 이미지에서 구조를 추출하는 OCR/구조화 도구다.
  다음 이미지는 한국어 책의 목차 페이지다.
  규칙:
  1) "파트" / "Part" / "제1부" / "1부" 단위를 parts[].title 로 추출. 없으면 전체를 parts[0]로 묶어라.
  2) "장" / "챕터" / "Chapter" 단위를 해당 파트의 sections[] 로 추출.
  3) 각 항목의 페이지 번호를 startPage/endPage 에 넣어라. endPage 는 다음 항목의 startPage - 1.
  4) 마지막 항목의 endPage 는 마지막으로 보이는 페이지 번호 또는 totalPages.
  5) 확신도 0~1 을 confidence 에 반환. 페이지 번호를 하나도 읽지 못하면 0.
  오직 위 JSON 스키마만 응답하라. 설명·코드블록 금지.
  ```
- 에러 핸들링:
  - API 키 없음 → 400 `{ error: "GEMINI_API_KEY missing" }`
  - 응답이 JSON 아니면 500 + raw 로깅 (서버 console만)
  - confidence < 0.4 → 200 응답하되 `warning: "low-confidence"` 포함 (UI에서 빨간 배지 표시)
- 재시도: 클라이언트 측에서만 3회 백오프 (T-35 상태 머신 내부)
**완료 기준**: 실제 책 목차 3권으로 테스트, `parts.length >= 1` 보장

```ts
// src/app/api/extract-toc/route.ts (발췌)
export async function POST(req: NextRequest) {
  const form = await req.formData();
  const images = form.getAll("image") as File[];
  const apiKey = process.env.GEMINI_API_KEY;
  if (!apiKey) return NextResponse.json({ error: "GEMINI_API_KEY missing" }, { status: 400 });

  const inlineData = await Promise.all(
    images.map(async (f) => ({
      inline_data: {
        mime_type: f.type || "image/jpeg",
        data: Buffer.from(await f.arrayBuffer()).toString("base64"),
      },
    })),
  );

  const body = {
    contents: [{ role: "user", parts: [{ text: PROMPT }, ...inlineData] }],
    generationConfig: { responseMimeType: "application/json", temperature: 0.1 },
  };

  const r = await fetch(
    `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=${apiKey}`,
    { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify(body) },
  );
  const json = await r.json();
  const text = json?.candidates?.[0]?.content?.parts?.[0]?.text ?? "{}";
  try {
    const parsed = JSON.parse(text);
    if ((parsed.confidence ?? 0) < 0.4) parsed.warning = "low-confidence";
    return NextResponse.json(parsed);
  } catch {
    console.error("[extract-toc] parse fail", text);
    return NextResponse.json({ error: "parse_failed" }, { status: 500 });
  }
}
```

```js
// scripts/build-icons.mjs
import sharp from "sharp";
import fs from "fs";
const SRC = "public/icon.svg";
const OUT = "public/icons";
fs.mkdirSync(OUT, { recursive: true });
for (const [name, size, pad] of [
  ["icon-192.png", 192, 0],
  ["icon-512.png", 512, 0],
  ["icon-maskable-512.png", 512, 0.2],   // 20% safe-zone
  ["../apple-touch-icon.png", 180, 0],
]) {
  const inner = Math.round(size * (1 - pad * 2));
  await sharp(SRC).resize(inner, inner)
    .extend({
      top: (size-inner)/2, bottom: (size-inner)/2,
      left: (size-inner)/2, right: (size-inner)/2,
      background: "#050505",
    })
    .png().toFile(`${OUT}/${name}`);
}
```
