- **주제**: Rutibooki MVP 기능 기획 — 북극성, 화면 플로우, 인터리빙, 정류장 메타포, 기록·회고, 4주 로드맵

## 1. 북극성 질문 — "이 앱이 진짜 뭘 해결하냐"

**김선배**: 혁준님이 "매일 책 한 권, 버튼 하나"라고 카피를 뽑았는데, 이게 해결하려는 건 **"책 읽기의 마찰력"**. 앱 열고 → 뭐 읽을지 고민 → 어디까지 읽었더라 → 아 귀찮다. 이 3초짜리 지옥.

**박아키**: 동의하는데 한 겹 더 있다. 단순한 마찰만의 문제가 아니라 **"선택 피로"** 와 **"진도 불안"** 이 섞여 있음. 책 7권 깔아놓고도 못 읽는 이유는 "오늘 내가 뭘 읽어야 맞는 건가"에 대한 확신이 없어서. 요일별 배정은 그래서 천재적. 결정을 앱이 대신 해줌.

**김선배**: 근데 거기서 한 발 더 나가면 위험. 인터리빙이니 AI 목차니 하다 보면 "결정을 대신 해주는 앱"이 아니라 "결정할 게 더 많은 앱"이 됨. 북극성은 하나로 잡아야 함.

> 🔴 **혁준님께 질문 1**
> "Rutibooki의 북극성 지표 1개를 고른다면 — **(A) 7일 연속 책 펼침률**, **(B) 일일 독서 지속 시간**, **(C) 주간 완독한 챕터 수** 중 뭐?"
> 이거에 따라 아래 모든 결정이 달라짐.

## 2. 화면 플로우 — "책 펼쳤어요" 다음 뭐가 나와야 하나

**박아키**: 혁준님은 "독서 타이머 화면"을 구상했는데, 타이머가 **독서의 본질을 왜곡**함. 타이머가 돌아가면 사람은 "빨리 끝내야지"가 됨.

**김선배**: 반대. 타이머 있어야 "아 15분만 해볼까"가 성립. 무한히 펼쳐진 시간은 오히려 진입 장벽. 포모도로가 왜 먹히는데.

**박아키**: 타이머 자체를 반대하는 게 아니라, **디폴트 화면이 타이머면 안 됨**.

```
┌─────────────────────────┐
│  지금 읽는 중            │
│                         │
│   《미움받을 용기》        │
│   2장: 모든 고민은        │
│         인간관계          │
│                         │
│   ───●───────           │  ← 현재 위치 점
│                         │
│  [ 여기서 쉴래요 ]         │ ← 1차 버튼
│  [ 기록 남기기 ]           │ ← 2차 버튼
│                         │
│       ⏱ 12:34           │ ← 타이머는 **작게**
└─────────────────────────┘
```

타이머는 있되, **구석에**. 주인공은 "내가 지금 어디 있나"여야 함.

**김선배**: 납득. 근데 "여기서 쉴래요" 누르면 어디로? 홈? 기록창?

**박아키**: 홈으로 바로. 기록은 선택사항. 기록 강요하면 그날로 이탈률 폭발.

**김선배**: 동의. **편안한 기록** = "안 써도 죄책감 없는 기록". 이모지 3개 중 하나 탭 정도.

```
오늘 어땠어요?
  🌱    🌊    🔥
 좋았어  그냥   불탔어
```

끝. 텍스트 입력 없음. 있으면 옵션으로만.

## 3. 인터리빙 클러스터 — 진짜 필요한가

**김선배**: 이거 **개발자만 흥미로운 기능**. 7권을 챕터별로 교차시키면 사용자는 "아 내가 지금 뭐 읽는 중이지?" 헷갈림. 학습 이론상으론 인터리빙이 좋지만, **앱 이탈률 기준으론 독약**.

**박아키**: 반은 맞고 반은 틀림. 이걸 **모드**로 두면 안 됨. 기본은 "매일 만나는 멘토(요일별 고정)", 인터리빙은 **"심화 모드"** 로 숨겨두기. 평소 안 보이다가, 유저가 2주 연속 완주했을 때 해금되는 식.

**김선배**: 해금이라는 단어 쓴 순간 이거 게임이 됨. 근데 그게 차라리 나음. 지금 단계에선 **빼는 게 맞다**는 거 합의.

**박아키**: MVP에서 빼고, 리텐션 측정 되고 나서 붙일지 결정.

> 🔴 **혁준님께 질문 2**
> "인터리빙 모드, **(A) MVP에서 완전히 빼기**, **(B) UI 숨겨두고 내부 플래그로만 실험**, **(C) 그래도 넣고 싶다** 중 어디?"

## 4. 버스 정류장 메타포

**박아키**: **세로 진행도 바를 가로 정류장 라인으로** 바꾸면 인지적으로 훨씬 직관적.

```
  ●────○────○────○────○
 1장   2장   3장   4장   끝
 완독  여기
```

**김선배**: 예쁘긴 한데, 책이 7권이면 이 라인이 7개 쌓임. 홈 화면 복잡해짐.

**박아키**: 홈에선 **오늘 요일 책만** 라인 표시. 나머지는 표지 썸네일만. 스와이프해서 요일 바꾸면 그때 그 책 정류장 나옴.

```
┌─────────────────────┐
│ 월요일 — 사피엔스      │
│                     │
│ ●━━●━━○━━○━━○━━○     │
│      ↑                │
│    지금 여기           │
│                     │
│  [ 책 펼쳤어요 → ]      │
└─────────────────────┘
```

**김선배**: 이건 좋다. 근데 **챕터 정보 어디서 가져와?** AI 목차 추출 얘기 나왔는데, GPT가 책 목차 제대로 줄 확률 생각보다 낮음. 환각 잘 함.

**박아키**: 맞음. **MVP는 수동 입력**이 맞고, AI 추출은 "편집기에서 한 번 뽑고 유저가 컨펌"하는 식으로 가야 함. 실시간 AI 금지.

## 5. 기록·회고 — 재방문 동기

**김선배**: 여기가 리텐션 핵심. 3주 차부터 "내가 이만큼 읽었네" 시각화 없으면 떠남.

**박아키**: **주간 회고 카드** 제안. 일요일 밤에 자동 생성.

```
┌──────────────────────┐
│  이번 주 루티부키 📖    │
│                      │
│  읽은 챕터   5개         │
│  만난 책     4권          │
│  기분 💚💚🌊🔥💚       │
│                      │
│  가장 오래 머문 책:     │
│  《미움받을 용기》       │
└──────────────────────┘
```

공유 가능하면 더 좋고. **강요 금지**. 공유 버튼은 작게.

**김선배**: 하이라이트/메모는 V2로. MVP에서 입력 UI 붙이면 배보다 배꼽이 커짐.

## 6. 실전 로드맵

**김선배**: 이번 주말에 하나만. **"책 펼쳤어요" 이후 화면**. 거기가 지금 가장 큰 구멍.

**박아키**: 동의. 그리고 챕터 데이터 구조 먼저 잡아야 정류장 UI가 가능. 2주차.

> 🔴 **혁준님께 질문 3**
> "챕터 데이터, **(A) 혁준님이 7권 직접 수동 입력(가장 빠름, 4시간)**, **(B) AI로 초안 뽑고 혁준님이 검수(하루)**, **(C) 유저가 직접 입력하는 UI 제공(일주일+)**, 어느 쪽?"

### Week 1~4 로드맵

| 주차 | 목표 | 구체 작업 | 완료 기준 |
|------|------|-----------|-----------|
| **W1** | 독서 화면 구축 | "책 펼쳤어요" → 독서 화면 / 작은 타이머 / "여기서 쉴래요" 버튼 | 홈 → 독서 → 홈 왕복 플로우 |
| **W2** | 정류장 UI + 챕터 데이터 | 챕터 스키마 설계 / 7권 수동 입력 / 가로 정류장 컴포넌트 | 현재 챕터 하이라이트 표시 |
| **W3** | 편안한 기록 | 이모지 3개 탭 / 로컬 저장 / 기록 없어도 진행 가능 | 일주일 분량 기록 누적 |
| **W4** | 주간 회고 카드 | 일요일 자동 생성 / 이미지 export / 공유(선택) | 주간 카드 1장 생성 완료 |

## 혁준님께 넘기는 결정 3가지

1. **북극성 지표 1개**: 연속 펼침률 / 독서 시간 / 완독 챕터 수 중?
2. **인터리빙 모드**: MVP 제외 / 숨겨둠 / 그래도 포함?
3. **챕터 데이터 입력**: 혁준님 수동 / AI 초안+검수 / 유저 입력 UI?

세 개 답 주면 W1 작업 바로 들어갈 수 있게 스펙 확정.

## 합의 사항 (두 페르소나 공통)

- 인터리빙 클러스터 MVP에서 **일단 뺀다** (리텐션 확인 후 재검토)
- 독서 화면 디폴트는 **타이머가 아니라 "지금 어디 있나"** (타이머는 구석)
- 기록은 **이모지 3개 탭이 최대치**, 텍스트는 옵션
- AI 목차는 **실시간 금지**, 편집기에서만
- 정류장 UI는 **오늘 요일 책에만** 표시 (7개 다 안 띄움)
